## Question 1: Basic Variables

- Your name (string)
- Your age (number)
- Whether you like programming (boolean)

## Question 2: Variable Scope

What will be logged in the following code?

```javascript
let x = 10;
{
    let x = 20;
    console.log(x);
}
console.log(x);
```

## Question 3: Variable Declaration

## Question 4: Variable Hoisting

```javascript

```
