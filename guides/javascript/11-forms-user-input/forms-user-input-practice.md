1. **Validation Issues**
   - Missing validation
   - Incorrect patterns
   - Poor error messages

2. **Form Handling**
   - Lost form data
   - Incorrect submission
   - Missing cleanup

3. **User Experience**
   - Poor feedback
   - Confusing errors
   - Missing guidance
