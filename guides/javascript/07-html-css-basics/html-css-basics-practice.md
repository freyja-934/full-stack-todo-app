## Common Issues to Watch For

1. **HTML Issues**
   - Invalid structure
   - Missing semantics
   - Poor accessibility

2. **CSS Issues**
   - Specificity conflicts
   - Missing fallbacks
   - Performance problems

3. **Layout Issues**
   - Broken responsiveness
   - Overflow problems
   - Alignment issues
