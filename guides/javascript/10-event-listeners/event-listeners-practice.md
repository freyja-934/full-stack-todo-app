## Common Issues to Watch For

1. **Memory Leaks**
   - Forgotten listeners
   - Missing cleanup
   - Event accumulation

2. **Event Bubbling**
   - Unexpected propagation
   - Missing preventDefault
   - Incorrect delegation

3. **Performance Issues**
   - Too many listeners
   - Missing debounce/throttle
   - Inefficient delegation
